Struktur folder repo

Supaya link di bawah langsung jalan, pakai struktur seperti ini:

index.html customer/index.html admin/index.html kurir/index.html

Kalau nama folder Anda berbeda, cukup ganti atribut href di tombol menu ini.

Customer

Halaman belanja online untuk customer. Cocok untuk guest checkout, hitung jarak otomatis, dan pemilihan kurir toko atau ekspedisi luar.

  • Katalog & keranjang
  • Checkout tanpa login ribet
  • Ambil lokasi GPS / pilih titik di peta
Masuk Customer Buka folder customer

Admin Pusat

Panel pusat untuk menerima order online, verifikasi pembayaran, assign kurir, input resi ekspedisi, dan memantau operasional pusat.

  • Filter internal / external
  • Verifikasi transfer
  • Tambah kurir & update status
Masuk Admin Buka folder admin

Kurir

Aplikasi kurir internal pusat untuk menerima order dekat toko, update status, simpan bukti kirim, dan mencatat COD serta pengeluaran harian.

  • Rute & status pengiriman
  • Bukti kirim & COD
  • Dompet kurir & closing kas
Masuk Kurir Buka folder kurir

Catatan upload

Upload file ini sebagai index.html di root repo. Lalu taruh file customer, admin, dan kurir di folder masing-masing seperti contoh di atas.

Catatan GitHub Pages

Setelah file masuk repo, aktifkan GitHub Pages dari branch main. Nanti halaman ini jadi pintu masuk utama untuk tes semua modul.
